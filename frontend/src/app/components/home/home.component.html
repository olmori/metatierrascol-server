Regulación

Esta web permite visualizar y descarar los datos de campo recogidos con la app de toma de datos en campo con el objetivo de que sirva de ayuda para la ayuda a la regularización de tierras en Colombia:

  • Muestra datos alfanuméricos sobre los predios.
  • Muestra la geometría de los predios sobre un mapa.
  • Muestra imágenes recogidas con la app sobre documentos o elementos del predio
  • Permite a los propietarios visualizar y compartir la información recogida.
  • Los datos se pueden enviar a los técnicos especialistas cuando se tiene internet
